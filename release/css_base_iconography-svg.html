Iconography (SVG)

  • Iconography Sizing (SVG)

npm install @npmcorp/pui-css-svg-iconography --save

We use Glyphish for scalable icons. They have been packaged up as [@npm/glyphish] for use, and inserting them as SVG <symbol> allows us to use the same icon multiple times with only a minimal bit of code for further instances.

These icons will always match the font color.

{{icon "@npm/glyphish/svg/g4/331-download" class="icon type-brand-8"}}

Iconography Sizing (SVG)

To vary the size of the icons, use the .icon-h1 through .icon-h6 classes, .icon-title, .icon-sm, or .icon-xs Don't use the typography size modifier classes (e.g. .title, .h1, .type-sm), as they may also imply other spacing changes.
